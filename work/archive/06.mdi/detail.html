  • Issue 1

    1. 이메일 전송

    - PHP 사용없이 이메일 전송 하기 위해서 emailJS 사용

                                            
/* 메 일 보 내 기 */ 
emailjs.init("user_oqJdpX5kj85Ks8cLOShCB");

window.onload = function () { 
    document.getElementById('contact-form').addEventListener('submit', function(event) { 
        event.preventDefault();
        this.contact_number.value = Math.random() * 100000 | 0;
        emailjs.sendForm('service_7oh32la', 'template_3d9wyhe', this); 
    });
}
                                        
$('#send-button').click(function () {
    if ($('#user_name').val() == '') { 
        alert('이름을 입력해 주세요'); $('#user_name').focus();
    } else if ($('#user_mail').val() == '') { 
        alert('메일을 입력해 주세요'); $('#user_mail').focus();
    } else if ($('#user_text').val() == '') { 
        alert('내용을 입력해 주세요'); $('#user_text').focus();
    } else {
        alert('메일이 전송되었습니다');
    } 
})

    2. 파비콘 제작 문제

    3. 영문판 제작

    - 클라이언트 측의 요청으로 영문 페이지 제작.

  • Issue 4

    4. https 도메인 문제

    - 결제된 도메인이 https를 지원하지 않아 https://mdimedical.net으로 접속 시 MDIMedical 페이지로 연결되지 않고 다른 페이지로 연결됨.
    추후 도메인을 결제할 경우 https를 지원하는 도메인으로 구매
